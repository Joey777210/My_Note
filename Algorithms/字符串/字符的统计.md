# 字符的统计
### 387.字符串中的第一个唯一字符
直接哈希表统计出现次数即可  
```
class Solution {
    public int firstUniqChar(String s) {
        if (s.length() == 0) {
            return -1;
        }

        Map<Character, Integer> map = new HashMap<>();
        for (int i = 0; i < s.length(); i++) {
            char c = s.charAt(i);
            map.put(c, map.getOrDefault(c, 0) + 1);
        }

        for (int i = 0; i < s.length(); i++) {
            if (map.get(s.charAt(i)) == 1) {
                return i;
            }
        }
        return -1;
    }
}
```

### 389.找不同
简单方法可以利用哈希表统计字符出现的次数  
**特殊方法：两个字符串中非新添加字符的出现次数总共都为偶数，而异或运算对同一个数字进行偶数次则为0，所以对两个字符串的每一个字符异或操作，最后剩下的则为新添加的字符**
```
class Solution {
    public char findTheDifference(String s, String t) {
        char ans = t.charAt(t.length() - 1);
        for (int i = 0; i < s.length(); i++) {
            ans ^= s.charAt(i);
            ans ^= t.charAt(i);
        }
        return ans;
    }
}
```

### 383.赎金信
没啥好说的，Map做字母统计即可，因为题目限定了26个小写字母，所以可以用数组代替  
```
class Solution {
    public boolean canConstruct(String ransomNote, String magazine) {
        int[] words = new int[26];
        for (int i = 0; i < magazine.length(); i++) {
            words[magazine.charAt(i) - 'a']++;
        }

        for (int i = 0; i < ransomNote.length(); i++) {
            words[ransomNote.charAt(i) - 'a']--;
            if (words[ransomNote.charAt(i) - 'a'] < 0) {
                return false;
            }
        }
        return true;
    } 
}
```
### 242.有效的字母异位词
跟上一题类似，小写字母统计，没啥好说的  
```
class Solution {
    public boolean isAnagram(String s, String t) {
        if (s.length() != t.length()) return false;
        int[] words = new int[26];
        for (int i = 0; i < s.length(); i++) {
            words[s.charAt(i) - 'a']++;
            words[t.charAt(i) - 'a']--;
        }

        for (int i : words) {
            if (i != 0) {
                return false;
            }
        }
        return true;
    }
}
```

### 49.字母异位词分组
https://leetcode-cn.com/problems/group-anagrams/solution/xiang-xi-tong-su-de-si-lu-fen-xi-duo-jie-fa-by--16/  
分组字母异位词，就要设定统一用来比较的key。这里将字符串中的字符排序作为key，拿到map中去比较。  
```
class Solution {
    public List<List<String>> groupAnagrams(String[] strs) {
        Map<String, List<String>> hash = new HashMap<>();
        for (String str : strs) {
            char[] chs = str.toCharArray();
            Arrays.sort(chs);
            String key = String.valueOf(chs); 
            //添加到对应的类中
            if (hash.containsKey(key)) {
                hash.get(key).add(str);
            } else {
                List<String> temp = new ArrayList<String>();
                temp.add(str);
                hash.put(key, temp);
            }
        }
        return new ArrayList<List<String>>(hash.values());
    }
}
```
也可以利用上一题中的字母数组，构建比较字符串map中的key  
```
public List<List<String>> groupAnagrams(String[] strs) {
    HashMap<String, List<String>> hash = new HashMap<>();
    for (int i = 0; i < strs.length; i++) {
        int[] num = new int[26];
        //记录每个字符的次数
        for (int j = 0; j < strs[i].length(); j++) {
            num[strs[i].charAt(j) - 'a']++;
        }
        //转成 0#2#2# 类似的形式
        String key = "";
        for (int j = 0; j < num.length; j++) {
            key = key + num[j] + '#';
        }
        if (hash.containsKey(key)) {
            hash.get(key).add(strs[i]);
        } else {
            List<String> temp = new ArrayList<String>();
            temp.add(strs[i]);
            hash.put(key, temp);
        }

    }
    return new ArrayList<List<String>>(hash.values());
}
```

### 451.根据字符出现频率排序
将字符串中的字符按照出现次数降序排列，用大顶堆，出现次数作为排序依据  
```
class Solution {
    public String frequencySort(String s) {
        char[] letter = new char[128];
        for (char c : s.toCharArray()) {
            letter[c]++;
        } 

        PriorityQueue<Character> heap = new PriorityQueue<>((a, b) -> letter[b] - letter[a]);

        for (int i = 0; i < letter.length; ++i) {
            if (letter[i] != 0) {
                heap.offer((char)i);
            }
        }

        // System.out.println(heap.toString());
        StringBuilder sb = new StringBuilder();
        while (!heap.isEmpty()) {
            char c = heap.poll();
            while(letter[c]-- > 0) {
                sb.append(c);
            }
        }       
        return sb.toString();
    }
}
```
### 423.从英文中重建数字
本题关键是发现很多数字的英文单词中有些独特的字母，并从这些字母入手逐一统计数字出现的次数  
```
class Solution {
    public String originalDigits(String s) {
        /*
        z  zero 
        one 
        w  two 
        three 
        u  four 
        five 
        x  six 
        seven 
        g  eight 
        nine
        */
        // 统计每个字母出现的次数
        int[] count = new int[26 + 'a'];
        for (char c : s.toCharArray()) {
            count[c]++;
        }

        // 根据独特字母推断数字的出现
        int[] out = new int[10];
        out[0] = count['z'];
        out[2] = count['w'];
        out[4] = count['u'];
        out[6] = count['x'];
        out[8] = count['g'];
        // h只有3和8
        out[3] = count['h'] - out[8];
        // f只有5和4
        out[5] = count['f'] - out[4];
        // s只有6和7
        out[7] = count['s'] - out[6];
        // i只有9 5 6 8
        out[9] = count['i'] - out[5] - out[6] - out[8];
        // n只有1 7，9中有两个
        out[1] = count['n'] - out[7] - 2 * out[9];

        StringBuilder output = new StringBuilder();
        for(int i = 0; i < 10; i++)
            for (int j = 0; j < out[i]; j++)
                output.append(i);
        return output.toString();
    }
}
```
### 657.机器人能否返回原点
L R对应，U D对应  
```
class Solution {
    public boolean judgeCircle(String moves) {
        int[] move = new int[2];
        for (char c : moves.toCharArray()) {
            switch (c) {
                case 'U':
                    move[0]++;
                    break;
                case 'D':
                    move[0]--;
                    break;
                case 'L':
                    move[1]++;
                    break;
                case 'R':
                    move[1]--;
                    break;
            }
        }
        // System.out.println(Arrays.toString(move));

        for (int i : move) {
            if (i != 0) return false;
        }
        return true;
    }
}
```
